Crate Buster

Link: Main Page

Tools used:
C, SDL, Git + Github, Aseprite, Logic Pro, Visual Studio

Project type:
Personal project


Project summary:
A 2D shoot-'em-up game made from scratch using only C and SDL.




Highlights:

Everything From Scratch

My primary goal for this project was to develop a game as from-scratch as possible. To that end, I wrote a simple game engine to support my development, with features like input handling, sprite sheets, and a collision system. On top of this engine, I programmed the game, implementing all gameplay features and UI myself. I also handled all of the game's art, audio and game design.

UI For Every User

Crate Buster features from-scratch UI widgets that can be navigated with gamepad, keyboard and mouse. UI was not originally intended to be a major focus of this project, but I ended up having so much fun putting together the widget system that I put a lot of time into making it nice.
